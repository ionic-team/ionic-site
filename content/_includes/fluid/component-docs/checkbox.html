Checkbox

Improve this Doc A checkbox is an input component that holds a *boolean* value. Checkboxes are no different than HTML checkbox inputs. However, like other Ionic components, checkboxes are styled differently on each platform. Use the `checked` attribute to set the default value, and the `disabled` attribute to disable the user from changing the value. _For more information, Check out the [API docs](../api/components/checkbox/Checkbox)._
