Modals

Improve this Doc Modals slide in off screen to display a temporary UI, often used for login or signup pages, message composition, and option selection. _For more information, Check out the [API docs](../api/components/modal/ModalController)._

Basic Usage

Demo Source ```typescript import { ModalController } from 'ionic-angular'; import { ModalPage } from './modal-page'; export class MyPage { constructor ( private modalController : ModalController ){ } presentModal() { let modal = this.modalController.create(ModalPage); modal.present(); } } ```
