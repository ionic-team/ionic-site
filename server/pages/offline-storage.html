Offline storage Features

Offline Storage offers a powerful, local database and performance-optimized query engine that enables fast, reliable offline experiences across mobile and desktop.

  • Local Filesystem Storage

    Data is stored directly on the filesystem outside of the browser environment.

  • NoSQL Query Engine

    A powerful query engine for building powerful, data-driven apps entirely in JavaScript.

  • On-Device Data Encryption

    Data is secured on the device using 256-bit AES full database encryption.

  • Peer-to-Peer Sync Support

    Synchronize data directly from device-to-device via peer-to-peer without retrieving cloud data.

  • Cross-Platform Deployment

    Works on mobile and desktop. Supported on iOS, Android, and Electron.

  • Couchbase Integration

    Connect and sync mobile data to your online Couchbases instances.
