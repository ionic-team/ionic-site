### Basic Usage

Demo Demo Source

To initialize Tabs, use [``](../api/components/tabs/Tabs/), with a child [``](../api/components/tabs/Tab/) for each tab: ```typescript import { Component } from '@angular/core'; import { Tab1 } from './tab1-page'; import { Tab2 } from './tab2-page'; @Component({ template: ` ` }) class MyApp { tab1: any; tab2: any; constructor() { this.tab1 = Tab1; this.tab2 = Tab2; } } ``` Individual tabs are just `@Components` ```typescript import { Component } from '@angular/core'; @Component({ template: ` Heart Tab 1` }) export class Tab1 { } @Component({ template: ` Star Tab 2` }) export class Tab2 { } ``` Notice that each `` binds to a `[root]` property, just like [``](../api/components/nav/Nav/) in the [Navigation](#navigation) section above. That is because each `` is really just a navigation controller. This means that each tab has its own history stack, and [`NavController instances injected`](../api/components/nav/NavController/#injecting_nav_controller) into children `@Components` of each tab will be unique to each tab: ```typescript import { Component } from '@angular/core'; import { NavController } from 'ionic-angular'; @Component({ ... }) class Tab1 { constructor(public navCtrl: NavController) { // Id is 1, nav refers to Tab1 console.log(this.nav.id) } } @Component({ ... }) class Tab2 { constructor(public navCtrl: NavController) { // Id is 2, nav refers to Tab2 console.log(this.nav.id) } } ```

Badges

Demo Demo Source

To add a badge to a tab, use the `tabBadge` and `tabBadgeStyle` attributes. The `tabBadgeStyle` attribute can be passed the name of any [color](/docs/theming/theming-your-app/): ```typescript @Component({ template: ` Tabs ` }) class TabBadgePage { constructor() { } } @Component({ template: ` ` }) export class BadgesPage { constructor() { this.tabOne = TabBadgePage; this.tabTwo = TabBadgePage; } } ``` For more information on tabs, check out the [Tabs API reference](../api/components/tabs/Tabs/).
