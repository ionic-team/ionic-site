Grid

Improve this Doc Ionic's grid system is based on [flexbox](http://www.w3.org/TR/css3-flexbox/), a CSS feature supported by all devices that Ionic supports. The grid is composed of three units — _grid_, _rows_ and _columns_. _For more detailed docs, Check out the [API docs](../api/components/grid/Grid)._ The grid system is made of a 12 columns, and each `ion-col` can be sized by setting the `col-` attribute. Demo Source ``` html This column will take 12 columns This column will take 6 columns ``` An `ion-col` can be sized for different breakpoints by setting the `ion--`. ``` html This column will be 12 columns wide by default, 9 columns at the small breakpoint, 6 at the medium breakpoint, 4 at large, and 3 at xl. ```
