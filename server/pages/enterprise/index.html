Ionic Framework

The open source UI Toolkit for building high performance mobile & PWAs with one codebase, for any device.

Truly cross-platform apps

The open source Ionic Framework is a complete app development kit for building cross-platform mobile and PWAs. Ionic’s rich library of UI components run and look beautiful on any platform or device, across mobile and desktop.

Built for web developers

Ionic apps leverage technology familiar to any web developer: HTML, CSS, JS. Your existing web talent can quickly and easily build apps using the skills they already know, without dependence on specialists and costly native developers.

Designed for the future

Ionic Framework is an open platform based on open web standards. Choose to build with React, Vue, and Angular, or any other front-end framework. Connect your apps to AWS, IBM, Google Firebase, or any backend cloud provider.

Ionic Pro

A fully integrated DevOps platform covering the entire app lifecycle for Enterprise teams building and shipping apps with Ionic.

Continuous Feedback. Push the right features at the right time, with fewer bugs.

Easy app sharing and testing provides realtime feedback on new features and updates. Continuous runtime monitoring helps to improve app quality by catching errors before your users do.

Continuous Delivery. Build, test, and release apps at the speed of development.

Automatic builds make it easy to test, share, or deploy updates each time a team member pushes code. Live Updates allow you to skip the app stores and send new features directly to users, as soon as they’re ready.

Secure Collaboration. Create amazing apps with the power of teams.

A shared dashboard and activity feeds keep team members informed on what’s happening. Granular permissions help safeguard apps and data, allowing only necessary team members to work in a given app.
