Basic Usage (Default)

Demo Source By default, all lists will be styled with divider lines: ```html {% raw %}{{item.title}}{% endraw %} ```

Basic Usage (No Lines)

Demo Source Adding the `no-lines` attribute will hide the dividers between list items: ```html {% raw %}{{item.title}}{% endraw %} ```

Inset List

Demo Source By default, lists have no outside margin, to add one, add the `inset` property to the list component. ```html {% raw %}{{item.title}}{% endraw %} ```

List Dividers

Demo Source To divide groups of items, use `` instead of ``. Use `` components to divide the group in to various sections: ```html A Angola Argentina B ```

List Headers

Demo Source Each list can include a header at the top of the list: ```html Action Terminator II The Empire Strikes Back Blade Runner ```

Icon List

Demo Source Adding [icons](#icons) to list items is a great way to hint about the contents of each item. The position of the icon can be set using the `item-left` and `item-right` attributes: ```html Herbology ```
