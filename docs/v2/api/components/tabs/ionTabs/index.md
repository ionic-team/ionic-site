<h1 class="api-title">

  ionTabs



</h1>





<p>Powers a multi-tabbed interface with a Tab Bar and a set of &quot;pages&quot;
that can be tabbed through.</p>
<p>Assign any tabs attribute to the element to define its look and feel.</p>
<p>For iOS, tabs will appear at the bottom of the screen. For Android, tabs
will be at the top of the screen, below the nav-bar. This follows each platform&#39;s
design specification, but can be configured with IonicConfig.</p>
<p>See the ionTab component&#39;s documentation for more details on individual tabs.</p>
